# usage

* count file bytes:

```shell
$ go run main.go -b test.txt
342190 test.txt
```

* count file lines:

```shell
$ go run main.go -l test.txt
7145 test.txt
```

* count file words:

```shell
$ go run main.go -w test.txt
58164 test.txt
```

* count file characters:

```shell
$ go run main.go -c test.txt
339292 test.txt
```
